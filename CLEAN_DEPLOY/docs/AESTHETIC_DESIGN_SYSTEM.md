**Rule:** Place key elements at φ-points (0.618 and 0.382 from edges)

**Applied to Documents:**
- Page headings: 0.382 from top
- Main content starts: 0.618 from top (after heading)
- Important diagrams: Centered at 0.618 × page width
- Pull quotes: 0.382 or 0.618 from left edge

### 2. TSP - Typographic Spacing Principles
**Rule:** Font sizes follow φ ratios

**Font Hierarchy:**
- **H1 (Titles):** Base × φ³ = Base × 4.236
