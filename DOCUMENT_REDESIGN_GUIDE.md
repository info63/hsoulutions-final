## TEMPLATE: φ-PERFECT PAGE DESIGN

### Page Specifications:

**Dimensions:**
- **Standard:** 8.5" × 11" (for academic submissions, must comply)
- **Ideal:** 8.5" × 13.75" (φ-ratio rectangle = 1.618:1)
- Use standard for papers, ideal for textbooks/posters

**Margins:**
```
Top: 0.618"
Bottom: 0.382"
Outer (right on odd pages, left on even): 0.618"
Inner (left on odd, right on even): 0.382"
```

**Content Area:**
- Width: 8.5" - (0.618 + 0.382) = 7.5"
- Height: 11" - (0.618 + 0.382) = 10" (standard page)
- Height: 13.75" - (0.618 + 0.382) = 12.75" (golden page)

**Grid System:**
- 13 columns (Fibonacci!)
- Column width: 7.5" ÷ 13 = 0.577" per column
- Gutter: 0.236" (= φ⁻²)

**Layout Options:**

**Option A: Single Column (Dense Text)**
```
|<-0.382->|<------- 7.5" Content ------->|<-0.618->|
```
Use for: Academic papers, legal documents

**Option B: Two Column (Balanced)**
```
|<-0.382->|<-- 4.6" Main -->|<- 2.9" Side ->|<-0.618->|
          |<---- 8 cols --->|<--- 5 cols -->|
          |   (61.5%)      |   (38.5%)    |
```
Use for: Textbooks, manuals, educational materials

**Option C: Three Column (Magazine Style)**
```
|<-0.382->|<2.9">|<2.9">|<1.7">|<-0.618->|
          | 5    | 5    | 3    |
          | Main | Main | Side |
```
Use for: Newsletters, infographics

## TYPOGRAPHY SYSTEM (TSP)

### Font Families:

**Headings:** Serif (Garamond, Georgia, Minion Pro)
- Elegant, traditional, respects history
- High readability for large text

**Body:** Sans-serif (Helvetica, Arial, Open Sans)
- Clean, modern, high readability for small text

**Code/Data:** Monospace (Courier, Consolas, Monaco)
- Fixed-width for tables, equations, code

### Font Sizes (φ-Based Hierarchy):

**Base Size: 11pt** (body text)

| Element | Formula | Size | Line Height |
|---------|---------|------|-------------|
| **H1 (Title)** | Base × φ³ | 46.6pt | 75.4pt |
| **H2 (Major Section)** | Base × φ² | 28.8pt | 46.6pt |
| **H3 (Subsection)** | Base × φ | 17.8pt | 28.8pt |
| **Body** | Base | 11pt | 17.8pt |
| **Caption** | Base × φ⁻¹ | 6.8pt | 11pt |
| **Fine Print** | Base × φ⁻² | 4.2pt | 6.8pt |

**Actual Sizes (Rounded to Standard Points):**
```
H1: 47pt
H2: 29pt
H3: 18pt
Body: 11pt
Caption: 7pt
Fine Print: 4pt
```

### Letter Spacing:
- H1: -0.02em (tighten for impact)
- H2: -0.01em
- H3: 0em
- Body: 0em
- Caption: +0.05em (open for readability)

### Line Length:
- Optimal: 55-65 characters per line (Fibonacci 55!)
- Single column: 65 characters max
- Two column: 45 characters per column

**Focal Points:**
- Horizontal: 0.618 × page width from left = 4.62" from left edge
- Vertical: 0.618 × page height from top = 6.8" from top (11" page)

**Key Elements to Place at φ-Points:**
- Page headings: 0.382 from top (4.2" down on 11" page)
- Main content starts: 0.618 from top (after heading)
- Important diagrams: Center at φ-point
- Pull quotes: 0.618 from left edge
- Signature lines: 0.618 from bottom

### Golden Rectangles:

**Create Boxes/Frames:**
```

Height = 5" ÷ 1.618 = 3.09"
```

Use for: Callout boxes, image frames, sidebars

### Golden Spiral Overlay:

**Why InDesign?**
- Professional publishing standard
- Precise φ-ratio controls

**Process:**
1. Create new document (8.5" × 13.75", margins as specified)

4. Import text from LaTeX/Markdown
5. Apply styles
6. Add visual elements (spirals, diagrams, color borders)
7. Export to PDF (PDF/A standard)

**Timeline per Document:**
- Simple (1-5 pages): 2 hours
- Medium (6-20 pages): 8 hours
- Complex (21+ pages): 20 hours
